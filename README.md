## Install

```
npm install --save wonderlangpark
```

Example Usage:
```js
const wonder = require("wonderlangpark");
console.log(wonder.towonder('디스코드는 갓입니다'))
// { input: '디스코드는 갓입니다.', output: '불화는 원더입니다.' }
```
